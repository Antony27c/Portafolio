# Notas terminal

# pwd
**Que hace?** Muestra la ruta completa de donde esta parado
**Comando exacto?**pwd
**Como lo use?** Para ver en que directorio estaba antes de crear el archivo

# ls
**Que hace?**Lista de todos los archivos del directorio actual
**Comando exacto?**ls
**Como lo use?**Para verificar si mis archivos index.html y style.css se crearon correctamente.

# cd
**Que hace?**Cambia la direccion actual, te moves entre carpetas
**Comando exacto?**cd
**Como lo use** Para entrar a la carpeta de mi  proyecto por la terminal.

# mkdir
**Que hace?**Crea una carpeta nueva
**Comando exacto?**mkdi tp1-mi-sitio
**Como lo use**Para crear la carpeta de mi proyecto al inicio del tranajo practico.

# Touch / New item
**Que hace?**Crea un archivo nuevo vacio
**Comando exacto** New-item Notas-terminal.md
**cuando la use** Para crear el archivo notas-terminal.md

# cat / get-content
**Que hace?**Muestra el contenido de un archivo en una terminal.
**Comando exacto?**Get-contet index.html
**Cuando lo use?**Para abrir el contenido de mi html sin abrirlo en el editor.

# Which / Get-command
**Que hace?**Muestra la ruta de dnde esta instalado un programa
**Comando exacto?**Get command git
**Cuando lo use?**Para verificar donde estabala instalado git

# code
**Que hace?**Abre el VS code desde la terminal
**Comando exacto?**code .
**Cuando lo use?**Para abrir la carpeta de mi proyecto directamente en VS code.

# clear
**Que hace?**Limpia la pantalla de la terminal
**Comando exacto?**clear
**Cuando lo use?**Para limpiar la terminal cuando tenia muchos comandos utilizados y queria verla mas ordenada

# echo
**Que hace?**Muestra un mensaje de texto en la terminal
**Comando exacto?**echo "Hola mundo"
**Cuando lo use?**para probar la terminal.
